# Communication-pattern-breakdown
Communication Pattern Breakdown breaks text into statements, evidence, actions, and communication patterns so readers can clearly see the structure of the information.

Foundations

Communication Pattern Breakdown is based on research from several established fields that study how information and persuasion appear in language.

Rhetoric and persuasion research
Studies how arguments and influence are structured, including credibility appeals, narrative framing, and emotional persuasion.

Discourse analysis and linguistics
Examines how language signals actions, roles, institutions, timelines, and relationships within text.

Information and media literacy
Provides methods for identifying claims, separating evidence from persuasion, and evaluating how information is presented.

These fields help identify structural patterns in communication such as:
	•	self-promotion
	•	authority framing
	•	abstraction or buzzword density
	•	mechanism absence
	•	narrative persuasion
	•	emotional framing

The tool uses these structural patterns to separate text into observable components so readers can better understand how information is being communicated.
How to Use
	1.	Copy a piece of text you want to examine.
This can include a profile description, post, article excerpt, application text, or report.
	2.	Paste the text into the tool.
	3.	The tool will organize the text into structural sections such as:

	•	plain language translation
	•	detected statements
	•	detected actions
	•	credentials and institutions
	•	timeline signals
	•	communication patterns
	•	claim–signal alignment

	4.	Review the structured output to see how the communication is organized and what signals appear in the language.

The goal is not to judge the text, but to make its structure easier to see.

Important Note

This tool does not determine whether information is true or false.

It surfaces structural patterns that appear in the language so readers can examine the communication more clearly. Final interpretation always requires human judgment
